1) **One-liner / Elevator pitch**
 «Эхо Несказанного» — это игра-исследование, где вы, перебирая вещи в опустевшей квартире дедушки, с помощью старой кинокамеры оживляете его воспоминания, чтобы разгадать тайну, которую он унес с собой.

2) **Core loop**
- **Исследование:** Игрок свободно перемещается по детализированной квартире (комнаты, коридор, балкон). Взаимодействует с предметами (книги, фотографии, старая мебель).
- **Погружение (Воспоминание):** Игрок может «проиграть» воспоминание. Мир вокруг меняется: квартира трансформируется в декорации прошлого. Игрок видит дедушку, его жену, детей, но не может с ними взаимодействовать — только наблюдать короткую сцену.
- **Сканирование (Анализ):** Найдя «мерцающий» предмет, игрок наводит на него кинокамеру. Камера анализирует предмет.

3) **Pillars**
- **Атмосфера:** Квартира — не просто декорация. Каждый предмет имеет значение.
- **Эмоциональный детектив:** Сюжет не рассказывается напрямую. Игрок — следователь, собирающий мозаику из обрывков сцен.
- **Минималистичное, но значимое взаимодействие:** Геймплей прост (ходишь, смотришь, слушаешь), но каждое взаимодействие двигает сюжет или углубляет понимание персонажей.

4) **Контент/ассеты** 
- **Локации:** Одна основная локация — квартира. 4-5 комнат. Для каждой комнаты нужно 2-3 состояния (разные временные эпохи).
- **3D-модели:** Модель главного героя, модель дедушки (в разных возрастах), модели жены и детей.
- **Анимации:** Анимации для сцен-воспоминаний. Анимации главного героя.
- **Текст:** Тексты дневника, описания предметов.
