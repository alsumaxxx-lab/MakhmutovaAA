- **Жанр:** Нарративное приключение / Симулятор ходьбы с элементами психологической драмы и головоломок.

- **USP:** Игрок исследует квартиру недавно ушедшего из жизни дедушки. Но вместо обычных вещей он находит «эхо» воспоминаний — интерактивные сцены из прошлого, которые позволяют собрать историю семьи, полную недомолвок и секретов. Главный инструмент — старая кинокамера, которая может записывать и проигрывать «эмоциональные частоты» предметов.
 
- **ЦА:** 14–35 лет, игроки, ценящие глубокие сюжеты.

- **Платформы:** PC (Windows, macOS).
